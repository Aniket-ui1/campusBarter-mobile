Check Frontend Token

Open your app at http://localhost:8083

Then open browser DevTools Console and paste:

// Check what token is being sent
fetch('http://localhost:8083', {credentials: 'include'})
  .then(() => {
    console.log('localStorage token:', localStorage.getItem('campusbarter_token'));
  });

// Try creating a listing manually
const token = localStorage.getItem('campusbarter_token');
console.log('Token:', token);

fetch('https://campusbarter-api-f3b4ascaemgthae3.canadacentral-01.azurewebsites.net/api/v1/listings', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}`
  },
  body: JSON.stringify({
    type: 'OFFER',
    title: 'Test from Console',
    description: 'Testing API call with stored token',
    credits: 1
  })
})
.then(r => r.json())
.then(data => console.log('Response:', data))
.catch(err => console.error('Error:', err));

Expected token format: mock-mock-user@sait.ca or mock-<userId>

If the token is NOT prefixed with mock-, that's your problem!
